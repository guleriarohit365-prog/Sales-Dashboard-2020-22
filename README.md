# 📊 Sales Dashboard (2020–2022)

## 1. Project Title / Headline
**Sales Analytics Dashboard – Sales Performance & Business Insights (2020–2022)**

An interactive Power BI dashboard developed to analyze sales performance, product demand, profit trends, customer transactions, and regional sales distribution across multiple categories and brands.

---

## 2. Short Description / Purpose

The Sales Dashboard is designed to provide a complete overview of business sales performance from **2020–2022**. It helps stakeholders monitor revenue, cost, profit, product demand, and geographical sales distribution for better decision-making and business growth.

---

## 3. Tech Stack

The dashboard was built using the following tools and technologies:

- 📊 **Power BI Desktop** – Dashboard design and interactive reporting  
- 🔄 **Power Query** – Data cleaning and transformation  
- 🧮 **DAX (Data Analysis Expressions)** – Measures and KPI calculations  
- 🗃️ **Excel / CSV Dataset** – Data source management  
- 🌍 **Map Visualization** – State-wise sales tracking  
- 📐 **Data Modeling** – Relationships and performance optimization  

---
